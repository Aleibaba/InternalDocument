# 任务种类查询

## 接口能力

| 接口名称   | 接口描述      | API地址  | 请求类型     |
| ------ | --------- | ------ | -------- |
| 任务种类查询 | 查询采集任务的种类 | /types | POST，GET |

## 任务种类查询接口

### 1. 接口描述
查询采集任务的种类

### 2. 请求参数

| 参数名称          | 是否必选 | 数据类型   | 参数位置   | 示例数据 | 描述   |
| ------------- | ---- | ------ | ------ | ---- | ---- |
| Authorization | 是    | string | Header |      | 用户名  |

### 3. 响应参数


| 参数名称     | 数据类型   | 描述     |
| -------- | ------ | ------ |
| typeId   | int    | 任务种类id |
| typename | String | 任务种类名称 |



### 4. 请求限制

| ---- |
| 1    |
